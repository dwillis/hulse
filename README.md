# Hulse

Hulse is a Ruby gem for accessing House and Senate roll call votes from the official sources on [house.gov](http://clerk.house.gov/evs/2013/index.asp) and [senate.gov](http://www.senate.gov/pagelayout/legislative/a_three_sections_with_teasers/votes.htm).

## Installation

Add this line to your application's Gemfile:

    gem 'hulse'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hulse
